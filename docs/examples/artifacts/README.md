# Measurement Artifact Envelope Examples

Honest illustrative examples of the canonical envelope shape defined in `docs/measurement-artifact-envelope.md`.

These files were authored as contract examples. **They are not captured outputs from live worker or executor endpoints.** Live endpoints still emit bespoke `output_payload` shapes (see Phase 2 doc §24 — Current implementation status). Adoption of the envelope by skills, exports, and the validator happens in later phases.

UUIDs and numeric values are synthetic. The shapes, field names, and enum values are authoritative.
