# Blueprint Importer v2 — Phase 5.5 CRM handoff contract examples

These files are **contract examples only**. They are NOT generated by any live
runtime, NOT inserted into the database, and NOT pushed to any CRM estimate.

UUIDs are fixed placeholders for diffability. The `deterministic_handoff_key`
and `deterministic_batch_key` values are illustrative — the live builder will
compute them via sha256 over the canonicalized inputs described in
`docs/blueprint-importer-phase-5-5-handoff-schema-contracts.md`.
