# Blueprint Importer v2 — Phase 1 contract examples

These files are **contract examples**, not live extraction output. They illustrate
the JSON shape that future phases will produce and persist. Phase 1 does not
extract, populate, or wire any of these objects.

UUIDs are fixed placeholders (`00000000-0000-0000-0000-0000000000xx`) so
examples remain deterministic and diffable.
